> 5、《金融情报机构间信息交换原则》 
``` bash
	• 埃格蒙特集团关于在金融情报机构层面上免费交换信息的原则应该建立在对等的基础上，包括信息的自动交换。
	• 金融情报机构管辖范围内对罪名定义的差异不应成为在金融情报机构层面免费交换信息的障碍。为此，金融情报机构的管辖范围应予以扩展至涵盖洗钱和恐怖融资活动的所有上游犯罪。
	• 金融情报机构之间的信息交换应该尽可能地采用非官方和便捷的方式，不得附加过多形式上的先决条件，同时要保证对隐私的保护以及共享数据的保密性。
	• 如果金融情报机构需要通过谅解备忘录进行信息交换，则应进行谈判并签署谅解备忘录，不得无故拖延。为了实现这一目标，金融情报机构有权独立签署谅解备忘录。
	• 金融情报机构之间的沟通可以直接进行，毋须通过中介。
	• 出于执法或司法目的，金融情报机构应该及时且最大限度地批准信息的分发。提供信息的金融情报机构不得拒绝信息的分发，除非此举超越了反洗钱和反恐融资活动的规定范围，妨碍刑事调查，与个人、法人或金融情报机构所在国的合法利益发生冲突，或者不符合所在国家法律的基本原则。任何拒绝批准的行为都应该给予适当的解释。
```

> 36、STR
``` bash
	在许多司法管辖区，要求向高级管理层或董事会报告关于可疑交易报告的特定信息。
	这些信息可能局限于所交报告数量、涉及的金额和合规人员观察到的重要趋向。（包括重大事项的详细补充资料的数值式概要，而不是所有可疑交易报告）
	在某些情况下，如果该活动对金融机构构成重大风险或潜在持续风险，机构领导应知情，以便高层就可能对制度、人员配备、产品、服务或机构维持的特定关系进行的调整作出决策。
	-- 所有可疑交易报告的数值式概要，以及关键可疑交易报告（包括政治敏感人士和关键员工）的概要
```

> 68、洗钱方法
``` bash
	MSB-不缺监管，但是制度薄弱，CDD重点是确认合法执照
	保险-缺乏对于中介的监管
	证券-制度薄弱，国际性，流速快

	赌场-处置离析
	贵金属珠宝艺术品-价值高、体积小、方便运输，虚假发票
	旅行社-高收机票，虚假订单，拆分交易
	交通工具经销-资金不上报，大量买卖构成复杂交易
	守门人-强制反洗钱义务
	投资和商品顾问-无关联账户转账撤回，非法所得为客户投资
	信托-少；信号（行为相关）：无证经营；标志（交易相关）：复杂公司安排
	房地产-泛滥；现金购买，掩饰受益人，升值

	自贸区-措施不足，缺少监管，程序薄弱，与海关缺乏合作
	贸易洗钱-高低发票，超载不足，空箱运输，空壳公司，重复开票，黑市比索

	预付卡-匿名，数量不限，全球取现，替代现金走私，发卡机构无需守法；措施：限制额度，与银行卡关联，存档记录，允许调查，制定国际标准
	信托-处于第一步（处置）；利用保密规则、防止查封；
	哈瓦拉-洗钱的任何阶段；缺少监管，无需保留记录；

	NPO-公众信任，大量资金来源，多使用现金，全球化。措施：说明用途，独立审计确保用于预期目的。
```

> 70、沃尔夫斯堡私人银行反洗钱准则
``` bash
	1、对拥有或曾经有用公共信托身份的个人进行EDD
	2、仅接受财富和资金来源具有合法性的客户
	3、账户受益人非客户本人时，必须识别所有账户的受益人身份
	4、对中介开展CDD，以确定该中间人的客户尽职调查流程是否令人满意
	5、新增客户和账户增加一人审批
	6、禁止使用集中账户

	补充，EDD：
	1、PEP
	2、居住或资金来源于高风险国家
	3、涉及洗钱活动或行业的个人
	4、与汇款企业、货币兑换所和转账代理等机构之间的业务关系，实施EDD
	5、高风险客户，或出于高风险行业的客户，以及类似地下钱庄/替代性汇款体系的活动
```

> 73、危险信号（特殊）
``` bash
	客户举止异常或过于紧张
	客户似乎有隐秘的事项或行为异常，如拒绝利用大量账户余额获得更高的利率
	交易涉及离岸机构，其名称与知名的合法金融机构相似
	交易涉及很难在地图上找到的陌生国家、地区或岛屿

	客户的永久地址在机构的服务区域之外
	客户的家庭或公司电话无法接通
	客户不愿接收账单或邮件

	客户存入大笔现金时不进行清点
	客户频繁使用小面额纸币兑换大面额纸币
	两名一起进入营业大厅的客户分别在不同的窗口办理低于报告限额的现金交易
	大额现金存款时存入大量大面额纸币
	同一天内在不同的分支机构进行大额现金交易
	主要以现金方式进行存取款交易，而不是采用支票转账的形式
	从他行提取现金后未经拆包即存入本行，金额巨大且频繁发生
	频繁使用现金购买金融票据
	进行外汇兑换交易的次数过多
	频繁开展外汇交易或货币互换，但从不关心利润

	存入大量第三方支票（经过背书的支票）

	频繁地通过电汇将所谓的国际业务利润转移到境外

	在保险箱服务区长时间逗留，可能表明保险箱内存有大量现金
	在使用保险箱后立即办理低于报告限额的现金存款
	租用多个保险箱

	使用一些引人注目但没有意义的术语，故意把交易变得复杂难懂
	要求信贷资金转给离岸公司，或者由离岸银行对贷款进行担保
	购买定期存单并作为质押申请贷款
	用现金存款作为质押申请贷款
	用离岸公司的现金作为质押申请贷款

	大型企业提供的财务报表不是由会计师所编制
	未针对支票而提取大量现金，表明其可能另有现金来源
	企业账户显示很少或几乎没有规律的定期活动

	在不提供合理依据的情况下，对信用证进行大幅修改；或者在付款前临时更改付款地点或受益人
	改变信用证中的付款地点，改为向非受益人所在地的某国账户支付
	备用信用证被用作投标或履约保函，但未说明标的项目或合同，或其受益人情况异常
	文件中缺少产权证书
	信用证来自洗钱风险极高的国家或地区
	交易使用重复修改或频繁延长的信用证

	客户使用投资账户作为转账工具将资金电汇至离岸地区
	客户希望通过一系列的小额交易来清算一笔大额头寸

	客户通过互联网或电话进行大量异常交易
	将个人账户用于商业目的

	员工规避定期休假

	两人或多人共同参与交易
	客户频繁前来购买金融票据，但每次金额均少于 3,000 美元（或当地记录保存限额）
	录入的交易关键词可能涉及可疑产品销售

	现金、不同来源的现金等价物（如银行本票或汇票）支付保单
	从国外支付保险费，特别是从离岸金融中心支付
	更改保单指定的受益人，新受益人与投保人之间的关系不明确
	赎回保险债券，但最初登记购买的个人和赎回的企业实体位于不同的国家或地区

	客户的账户有原因不明或突然增多的电汇
	客户账户存在原因不明的大量交易活动，但证券交易量很小

	借款人 / 买家为主要住所申请贷款，但并不住在贷款申请上填写的新主要住所
	买家或卖家在抵押贷款交易中的代理人未取得经营执照

	购买量或进口量远远超过预期销量
	珠宝店（零售）出售金条、钱币和散钻
	在进口 / 出口资金收款 / 转账中，订购客户 / 受益人为货币服务企业

	经常进行舍入金额或整数金额的交易

	客户账户似乎是流入账户，办理现金存款（通常低于 10,000 美元的报告限额）的城市 / 州不是客户居住地或业务所在地。在流入账户的情况下，资金通常在存入后被迅速取出（同一日）
	从可能的流入账户存入的支票似乎是预签，签字栏和付款人栏的笔迹不同

	恐怖融资：
	1、行为标志：
	使用虚假公司，包括空壳公司
	滥用非营利组织

	2、交易标志：
	在国内外 ATM 上频繁操作
	使用多个外国银行账户




	• 客户身份识别；
	• 风险管理；
	• 客户接纳；和
	• 高风险账户监控。
```
